# Native-сборка в Android (NDK/JNI)

## Что это такое?

Native-сборка — это компиляция C/C++ кода в машинные инструкции (`.so` файлы), которые выполняются напрямую процессором, минуя виртуальную машину Android (ART).

## Архитектура процесса

```
┌─────────────────────────────────────────────────────────────────┐
│                      Android Приложение                          │
├─────────────────────────────────────────────────────────────────┤
│  Kotlin/Java код          │         C/C++ код (Native)          │
│  ┌─────────────────┐      │      ┌─────────────────────────┐    │
│  │  MainActivity   │      │      │   whisper.cpp           │    │
│  │  ViewModel      │      │      │   onnxruntime           │    │
│  │  SpeechRecognizer│──────┼──────│   sherpa-tts            │    │
│  └─────────────────┘  JNI │      └─────────────────────────┘    │
│         │                 │                 │                    │
│         │                 │                 ▼                    │
│         ▼                 │      ┌─────────────────────────┐    │
│  ┌─────────────────┐      │      │   .so библиотеки        │    │
│  │   Kotlin код    │      │      │   libwhisper-jni.so     │    │
│  │   вызывает      │      │      │   libsherpa-jni.so      │    │
│  │   native методы │      │      │   libonnxruntime.so     │    │
│  └─────────────────┘      │      └─────────────────────────┘    │
│                           │                 │                    │
│                           │                 ▼                    │
│                           │      ┌─────────────────────────┐    │
│                           │      │   ARM64 / ARMv7 / x86   │    │
│                           │      │   (машинный код)        │    │
│                           │      └─────────────────────────┘    │
└─────────────────────────────────────────────────────────────────┘
```

## JNI (Java Native Interface)

JNI — это мост между Kotlin/Java и C/C++ кодом.

## Требования для сборки

1. **Android NDK** — установить через SDK Manager
2. **CMake** — установить через SDK Manager  
3. **Min SDK 26+** — для поддержки C++17
